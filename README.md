# BB Floating Avatar Viewer

Расширение для SillyTavern, которое открывает аватары персонажей и персоны в отдельных плавающих окнах.

## Возможности

- Открывает аватар по клику в отдельном окне поверх интерфейса.
- Поддерживает аватары персонажей, сообщений, групп и пользовательской персоны.
- Можно открыть несколько окон одновременно.
- Окна можно перетаскивать, увеличивать и уменьшать.
- Поддерживается масштабирование колесом мыши.
- На мобильных устройствах работают жесты, включая pinch-to-zoom.
- Размер и положение окна могут запоминаться между открытиями.
- Клавиша `Escape` закрывает верхнее активное окно.

## Настройки

- Перехват кликов по аватарам можно включить или выключить.
- Запоминание положения и размера можно включить или выключить.
- Для ручного вызова доступен API в `window.bbFloatingAvatarViewer.openUrl(src, label)`.

## Установка

1. Откройте SillyTavern.
2. Перейдите в `Расширения -> Установить расширение`.
3. Вставьте ссылку на репозиторий:
   `https://github.com/maxkara14/BB-Floating-Avatar-Viewer`
4. Перезагрузите страницу.
5. Кликните по аватару персонажа или персоны.

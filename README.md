# Floating Avatar Viewer

## Что умеет

- Открывает аватарку в отдельном окне.
- Можно открывать одновременно аватарку персоны и аватарку бота.
- Аватарку можно двигать, увеличивать или уменьшать.

## Установка

1. Откройте **SillyTavern**.
2. Перейдите в раздел **Extensions / Расширения**.
3. Выберите установку расширения по репозиторию.
4. Укажите URL этого репозитория:  `https://github.com/maxkara14/BB-Floating-Avatar-Viewer`
5. Перезагрузите страницу SillyTavern.
6. Наслаждайтесь!
